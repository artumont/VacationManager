Welcome to Vacation Manager

This is a simple vacation manager UI built with Angular and Tailwind CSS

Employee UIs

/employee/request-vacation

This page is used by employees to request vacations

/employee/ongoing-vacations

This page shows current and upcoming vacations

/employee/past-vacations

This page shows the history of past vacations

Manager UIs

/manager/approve-vacations

This page is used by managers to approve vacations

/manager/manage-vacations

This page is used by managers to manage team vacations

Admin UIs

/admin/manage-permissions

This page is used by admins to manage permissions

/admin/configure-holidays

This page is used by admins to configure company holidays
